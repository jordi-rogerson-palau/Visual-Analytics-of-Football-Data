Treball de final del Grau en Ciència i Enginyeria de Dades a la UPC.

ETL de dades obertes de Statsbomb per a crear un aplicació web amb Streamlit on s'hi puguin visualitzar charts relacionats amb la temporada de futbol 2015/2016, de les 5 grans lligues Europees, per a poder resoldre unes qüestions plantejades principalment en relació amb la reputació de cada una de les lligues.

Gairebé tots els partits estan disponibles, menys 3 de la "Ligue 1". 

**pages** conté els scripts corresponents a les pàgines de l'aplicació web.

**table creations** conté totes les extraccions de Statsbomb i algun script per a netejar les taules creades, que he guardat en un DUckDB local.
